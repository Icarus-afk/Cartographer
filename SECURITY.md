# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Cartographer, please report it privately.

**Do not open a public issue.** Instead, send an email to <ehasan.ahmed01@gmail.com>.

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (if known)

You will receive a response within 48 hours. We will keep you informed as the issue is investigated and resolved.

## Scope

The following are in scope:
- The `cartographer` Python package and its dependencies
- The VS Code extension (`editors/vscode/`)
- The MCP server and protocol implementation
